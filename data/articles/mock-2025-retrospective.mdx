## 성장 영역

### 프론트엔드

```typescript
// TypeScript strict mode 적용
interface User {
  id: number;
  name: string;
  email: string;
}

function processUser(user: User): string {
  return `${user.name} (${user.email})`;
}
```

TypeScript strict mode를 처음 적용했을 때 많은 타입 에러를 만났지만, 이를 통해 더 안전한 코드를 작성하게 되었습니다.
