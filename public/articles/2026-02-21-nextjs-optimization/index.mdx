## 기본 사용법

Next.js Image 컴포넌트는 다음과 같이 사용합니다:

```jsx
import Image from "next/image";

export default function Hero() {
  return (
    <Image
      src="/images/hero.webp"
      alt="Hero image"
      width={1200}
      height={600}
      priority
    />
  );
}
```

## 주요 기능

Image 컴포넌트를 사용하면 자동으로:

- **차세대 형식 제공**: WebP, AVIF 등 최신 형식 제공
- **반응형 이미지**: 다양한 화면 크기에 맞게 자동 조정
- **Lazy Loading**: 뷰포트 진입 시에만 로드
- **플레이스홀더**: 로딩 중 사용자 경험 개선

성능 최적화로 Lighthouse 점수를 높일 수 있습니다.
